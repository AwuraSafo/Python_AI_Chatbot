# Python AI Chatbot

This is a team-based Python AI chatbot prototype designed to experiment with
humor, conversation patterns, and learning behaviors in intelligent systems.

## Features
- Responds to user input with context-aware answers
- Includes humorous responses inspired by human-AI interactions
- Easy to extend with additional questions and answers

## Technologies Used
- Python 3
- Natural Language Toolkit (NLTK) [optional]
- Simple text processing

## How to Run
1. Clone the repository
2. Install dependencies: `pip install -r requirements.txt`
3. Run: `python chatbot.py`
